== README

Golfstats
* Ruby version 2.3.0

Golfstats is a golf stat tracking applicaton created in Ruby.

Here are some of the key features -

Devise (authentication)
Users linked to Rounds
Rounds track -
* course
* score
* fairways hit
* greens hit
* putting

Views contain a dashboard that keeps track of the users stats.  Users can edit and modify rounds.

There is also a section for improvement that links to videos (not user uploaded)
